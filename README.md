#  Banking Management System V2

A console-based Banking Management System built using Python. This project demonstrates Object-Oriented Programming (OOP), file handling, JSON data storage, authentication, and core banking operations.

## Features

* Create New Bank Account
* Secure Login Authentication
* Deposit Money
* Withdraw Money
* Transfer Money Between Accounts
* Change Password
* Check Account Balance
* Transaction History Tracking
* Password Verification for Sensitive Operations
* Persistent Data Storage using JSON

##  Technologies Used

* Python
* Object-Oriented Programming (OOP)
* JSON
* File Handling
* Exception Handling

##  How to Run

### Clone the Repository

```bash
git clone https://github.com/Gautam2-468/Banking-Management-System-V2.git
```

### Navigate to the Project Directory

```bash
cd Banking-Management-System-V2
```

### Run the Program

```bash
python banking_management.py
```
